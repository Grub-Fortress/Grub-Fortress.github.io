Installation Instructions

  1. Download and extract the ZIP file above to your preferred location.
  2. Copy the extracted folder into your SourceMods directory. Typically located at:
    C:\Program Files (x86)\Steam\steamapps\sourcemods\
  3. If there is a Patch available, make sure to extract that on top of the version the patch is meant for
  4. Restart Steam. "Grub Fortress" should now appear in your games library.
  5. Launch and enjoy!

Requirements

  • Steam with Source SDK Base 2013 Multiplayer installed
  • Windows 7 or newer (Linux compatibility not tested)
  • ~2GB disk space

Troubleshooting

If the game doesn't appear in Steam after installing, make sure:

  • You extracted the mod into the correct sourcemods folder
  • Steam was restarted after extraction

If the game doesn't launch, make sure:

  • You have the correct beta branch of Source SDK Base 2013 Multiplayer selected (None)
